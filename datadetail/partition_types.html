PC Partition Type Identifiers

This table contains partition identifiers (system indicators) for various operating systems, file systems, boot managers, and other utilities used on PC systems.

ID Name Description
00 Empty Not used to designate unused area on the disk, but marks an unused partition table entry. All other fields should be zero as well.
01 DOS 12-bit FAT DOS family of single-user operating systems for PCs. Type 01 is for partitions smaller than 16 MB.
02 XENIX root Xenix is an old port of Unix V7. Microsoft Xenix OS was announced August 1980.
03 XENIX /usr Xenix /usr partition for the old Unix V7 port.
04 DOS 3.0+ 16-bit FAT (up to 32M) DOS 3.0+ 16-bit FAT partition up to 32MB. Some old DOS versions require this partition to be in the first physical 32 MB.
05 DOS 3.3+ Extended Partition Supports at most 8.4 GB disks. Extended partition containing a linked list of logical partitions.
06 DOS 3.31+ 16-bit FAT (over 32M) DOS 3.31+ 16-bit FAT partition over 32MB. Partitions are at most 2 GB for DOS and Windows 95/98.
07 OS/2 IFS (e.g., HPFS) Installable File System. Also used for Windows NT NTFS, exFAT, Advanced Unix, and QNX2.x pre-1988.
08 OS/2 (v1.0-1.3 only) Also used for AIX boot partition, SplitDrive, Commodore DOS, DELL partition spanning multiple drives, and QNX 1.x and 2.x.
09 AIX data partition IBM's version of Unix. Also used for Coherent filesystem and QNX 1.x and 2.x.
0a OS/2 Boot Manager OS/2 Boot Manager. Also used for Coherent swap partition and OPUS (Open Parallel Unisys Server).
0b WIN95 OSR2 FAT32 Windows 95 OSR2 FAT32 partitions up to 2047GB.
0c WIN95 OSR2 FAT32, LBA-mapped Extended-INT13 equivalent of 0b.
0d SILICON SAFE Planned by siliconsafe.
0e WIN95: DOS 16-bit FAT, LBA-mapped Windows 95 DOS 16-bit FAT with LBA mapping.
0f WIN95: Extended partition, LBA-mapped Windows 95 Extended partition with LBA mapping. Extended-INT13 equivalent of 05.
10 OPUS (?) Maybe decimal, for type 0a.
11 Hidden DOS 12-bit FAT OS/2 Boot Manager hides primary DOS partitions by changing ID. Also Leading Edge DOS 3.x logically sectored FAT.
12 Configuration/diagnostics partition Used by Compaq for configuration utility partition, hibernation partition, NCR diagnostics, Intel Service Partition, and IBM Rescue and Recovery.
14 Hidden DOS 16-bit FAT <32M Hidden DOS 16-bit FAT partition less than 32MB. Also AST DOS with logically sectored FAT.
16 Hidden DOS 16-bit FAT >=32M Hidden DOS 16-bit FAT partition 32MB or larger.
17 Hidden IFS (e.g., HPFS) Hidden Installable File System (e.g., HPFS).
18 AST SmartSleep Partition Ascentia laptops Zero Volt Suspend Partition or SmartSleep Partition of size 2MB+memory size.
19 Unused Claimed for Willowtech Photon coS (completely optimized system).
1b Hidden WIN95 OSR2 FAT32 Hidden Windows 95 OSR2 FAT32 partition.
1c Hidden WIN95 OSR2 FAT32, LBA-mapped Hidden Windows 95 OSR2 FAT32 with LBA mapping. Sometimes a hidden IBM rescue partition.
1e Hidden WIN95 16-bit FAT, LBA-mapped Hidden Windows 95 16-bit FAT with LBA mapping.
20 Unused Rumoured to be used by Willowsoft Overture File System (OFS1).
21 Reserved HP Volume Expansion, SpeedStor variant. Also claimed for FSo2 (Oxygen File System).
22 Unused Claimed for Oxygen Extended Partition Table.
23 Reserved Reserved partition type.
24 NEC DOS 3.x NEC MS-DOS 3.30 logically sectored FAT. MBR could have up to 8 partition entries.
26 Reserved Reserved partition type.
27 PQservice Acer laptop hidden rescue partition (FAT32 or NTFS). Also Windows RE hidden partition, MirOS partition, and RouterBOOT kernel partition.
2a AtheOS File System (AFS) AtheOS/Syllable OS filesystem, an extension of BeFS.
2b SyllableSecure (SylStor) A variation on AthFS with added security.
31 Reserved Reserved partition type.
32 NOS Operating system being developed by Alien Internet Services in Melbourne Australia.
33 Reserved Reserved partition type.
34 Reserved Reserved partition type.
35 JFS on OS/2 or eCS OS/2 Warp Server for e-Business and eComStation implementation of JFS (IBM AIX Journaling Filesystem).
36 Reserved Reserved partition type.
38 THEOS ver 3.2 2gb partition THEOS multiuser multitasking OS version 3.2 2GB partition.
39 Plan 9 partition Plan 9 operating system partition. Also THEOS ver 4 spanned partition.
3a THEOS ver 4 4gb partition THEOS version 4 4GB partition.
3b THEOS ver 4 extended partition THEOS version 4 extended partition.
3c PartitionMagic recovery partition PowerQuest PartitionMagic recovery partition used during disk operations.
3d Hidden NetWare Hidden Novell NetWare partition.
40 Venix 80286 Very old Unix-like operating system for PCs. Also PICK multi-user operating system.
41 Linux/MINIX (sharing disk with DRDOS) Old recommendation for Linux/MINIX on disks shared with DRDOS. Also Personal RISC Boot and PPC PReP Boot.
42 Linux swap (sharing disk with DRDOS) Linux swap partition on disks shared with DRDOS. Also SFS (Secure Filesystem) and Windows 2000 dynamic extended partition marker.
43 Linux native (sharing disk with DRDOS) Linux native partition on disks shared with DRDOS.
44 GoBack partition GoBack utility partition that records disk changes for rollback capability.
45 Boot-US boot manager Boot-US boot manager partition. Also Priam and EUMEL/Elan.
46 EUMEL/Elan EUMEL/Elan multiuser multitasking system partition.
47 EUMEL/Elan EUMEL/Elan multiuser multitasking system partition.
48 EUMEL/Elan EUMEL/Elan multiuser multitasking system partition.
4a Mark Aitchison's ALFS/THIN lightweight filesystem for DOS ALFS/THIN lightweight filesystem. Also AdaOS Aquila (Withdrawn).
4c Oberon partition Aos (now A2) filesystem partition for Oberon operating system.
4d QNX4.x QNX4.x first partition.
4e QNX4.x 2nd part QNX4.x second partition.
4f QNX4.x 3rd part QNX4.x third partition. Also Oberon partition for Nat filesystem.
50 OnTrack Disk Manager (older versions) RO OnTrack Disk Manager read-only partition. Also Lynx RTOS and Native Oberon.
51 OnTrack Disk Manager RW (DM6 Aux1) OnTrack Disk Manager read-write partition. Also Novell.
52 CP/M CP/M operating system. Also Microport SysV/AT.
53 Disk Manager 6.0 Aux3 OnTrack Disk Manager 6.0 Auxiliary partition 3.
54 Disk Manager 6.0 Dynamic Drive Overlay (DDO) OnTrack Disk Manager 6.0 Dynamic Drive Overlay.
55 EZ-Drive MicroHouse EZ-Drive disk manager partition.
56 Golden Bow VFeature Partitioned Volume Non-Standard DOS Volume. Also DM converted to EZ-BIOS and AT&T MS-DOS 3.x logically sectored FAT.
57 DrivePro StorageSoft DrivePro partition. Also VNDI Partition.
5c Priam EDisk Priam EDisk Partitioned Volume (Non-Standard DOS Volume).
61 SpeedStor Storage Dimensions SpeedStor Volume (Non-Standard DOS Volume).
63 Unix System V (SCO, ISC Unix, UnixWare, ...) Unix System V, Mach, GNU Hurd. UnixWare 7.1 partition must start below 4GB limit.
64 PC-ARMOUR protected partition PC-ARMOUR disk protection partition. Also Novell Netware 286, 2.xx.
65 Novell Netware 386, 3.xx or 4.xx Novell Netware 386 partition using Turbo FAT filesystem.
66 Novell Netware SMS Partition Novell Netware Storage Management Services partition (no longer used).
67 Novell Novell partition (Wolf Mountain).
68 Novell Novell partition.
69 Novell Netware 5+, Novell Netware NSS Partition Novell Netware 5+ NSS (Novell Storage Services) partition.
6e ?? Unknown partition type, reported once.
70 DiskSecure Multi-Boot DiskSecure Multi-Boot partition.
71 Reserved Reserved partition type.
72 V7/x86 V7/x86, a port of UNIX Version 7 to the PC.
73 Reserved Reserved partition type.
74 Reserved Reserved partition type. Also Scramdisk partition for disk encryption software.
75 IBM PC/IX IBM PC/IX partition.
76 Reserved Reserved partition type.
77 M2FS/M2CS partition M2FS/M2CS partition. Also VNDI Partition (not in actual use).
78 XOSL FS XOSL Bootloader filesystem.
7e Unused Claimed for F.I.X.
7f Unused Proposed for the Alt-OS-Development Partition Standard.
80 MINIX until 1.4a MINIX operating system until version 1.4a.
81 MINIX since 1.4b, early Linux MINIX since version 1.4b and early Linux. Also Mitac disk manager.
82 Prime Prime partition. Also Solaris x86 and Linux swap.
83 Linux native partition Linux native partition supporting various filesystem types (ext2, ext3, reiserfs, etc.).
84 OS/2 hidden C: drive OS/2-renumbered type 04 partition. Also Hibernation partition for laptops.
85 Linux extended partition Linux extended partition.
86 Old Linux RAID partition superblock Old Linux RAID partition. Also FAT16 volume set (Legacy Fault Tolerant).
87 NTFS volume set Legacy Fault Tolerant NTFS volume or HPFS Fault-Tolerant mirrored partition.
88 Linux plaintext partition table Linux plaintext partition table.
8a Linux Kernel Partition (used by AiR-BOOT) Linux Kernel Partition used by AiR-BOOT boot loader.
8b Legacy Fault Tolerant FAT32 volume Legacy Fault Tolerant FAT32 volume.
8c Legacy Fault Tolerant FAT32 volume using BIOS extd INT 13h Legacy Fault Tolerant FAT32 volume using BIOS extended INT 13h.
8d Free FDISK 0.96+ hidden Primary DOS FAT12 partitition Free FDISK hidden Primary DOS FAT12 partition.
8e Linux Logical Volume Manager partition Linux LVM (Logical Volume Manager) partition.
90 Free FDISK 0.96+ hidden Primary DOS FAT16 partitition Free FDISK hidden Primary DOS FAT16 partition.
91 Free FDISK 0.96+ hidden DOS extended partitition Free FDISK hidden DOS extended partition.
92 Free FDISK 0.96+ hidden Primary DOS large FAT16 partitition Free FDISK hidden Primary DOS large FAT16 partition.
93 Hidden Linux native partition Hidden Linux native partition. Also Amoeba.
94 Amoeba bad block table Amoeba distributed operating system bad block table.
95 MIT EXOPC native partitions MIT EXOPC native partitions.
96 CHRP ISO-9660 filesystem CHRP (Common Hardware Reference Platform) ISO-9660 filesystem.
97 Free FDISK 0.96+ hidden Primary DOS FAT32 partitition Free FDISK hidden Primary DOS FAT32 partition.
98 Free FDISK 0.96+ hidden Primary DOS FAT32 partitition (LBA) Free FDISK hidden Primary DOS FAT32 partition with LBA. Also Datalight ROM-DOS Super-Boot Partition.
99 DCE376 logical drive Mylex DCE376 EISA SCSI adaptor logical drive for partitions beyond 1024th cylinder.
9a Free FDISK 0.96+ hidden Primary DOS FAT16 partitition (LBA) Free FDISK hidden Primary DOS FAT16 partition with LBA.
9b Free FDISK 0.96+ hidden DOS extended partitition (LBA) Free FDISK hidden DOS extended partition with LBA.
9e ForthOS partition ForthOS partition (Forth operating system). Also older VSTA.
9f BSD/OS Current BSD/OS partition type (BSDI).
a0 Laptop hibernation partition Hibernation partition for various laptops (IBM Thinkpad, Phoenix NoteBIOS, Toshiba, Sony VAIO).
a1 Laptop hibernation partition Save-to-Disk partition on NEC 6000H notebook. Also HP Volume Expansion (SpeedStor variant).
